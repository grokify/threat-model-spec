## [v0.1.0] - 2026-03-01

### Highlights

- JSON-based intermediate representation for threat modeling diagrams with D2 and STIX 2.1 export
- Comprehensive security framework mappings: MITRE ATT&CK, MITRE ATLAS, OWASP Top 10 (API/LLM/Web), CWE, CVSS, STRIDE

### Added

- JSON IR schema for threat modeling diagrams (DFD, Attack Chain, Sequence)
- `ir` package with types for elements, boundaries, flows, attacks, actors, and messages
- `ir/mappings.go` with framework mapping types for MITRE ATT&CK, ATLAS, OWASP, CWE, CVSS, and STRIDE
- `diagram` package for programmatic diagram construction with D2 rendering
- `stix` package for STIX 2.1 export (Bundle, AttackPattern, ThreatActor, Infrastructure, Indicator, Vulnerability)
- `stride` package with STRIDE threat category types and color mappings
- `killchain` package with MITRE ATT&CK tactics and Lockheed Martin Cyber Kill Chain phases
- `tms` CLI with `generate`, `validate`, and `version` subcommands
- `tms generate --stix` flag for STIX 2.1 JSON export
- `tms generate -svg` flag to render D2 diagrams to SVG
- D2 style library with classes for STRIDE threats, trust boundaries, DFD elements, and attack flows
- OpenClaw WebSocket vulnerability example with DFD, attack chain, and sequence diagrams
- Demo vulnerable server and malicious page for educational attack demonstration
- Validation for diagram type-specific field requirements

### Documentation

- README with installation, quick start, and framework mapping reference
- OpenClaw article.html with comprehensive vulnerability analysis and OWASP mappings
- D2 style reference for STRIDE threat annotations and trust boundaries
