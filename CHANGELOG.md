## [v0.4.0] - 2026-04-19

### Highlights

- Versioned specification structure following OpenAPI pattern (`versions/v0.4.0/`)
- Comprehensive security lifecycle: mitigations, detections, and response actions
- Attack tree diagrams with AND/OR logic gates
- LINDDUN privacy threat framework with D2 rendering
- Control framework mappings: NIST CSF, CIS Controls v8, ISO 27001
- Compliance framework references: SOC 2, PCI-DSS, HIPAA, GDPR, FedRAMP
