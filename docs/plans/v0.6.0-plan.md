# Threat Model Spec v0.6.0 Enhancement Plan

> **Status:** Complete
>
> **Goal:** Add OWASP Agentic Security (ASI) support and OWASP reference data

## Overview

The OpenClaw vulnerability analysis revealed gaps in the current spec:

1. **No ASI support** — The new OWASP Top 10 for Agentic Applications (2026) isn't supported
2. **No OWASP reference data** — Framework IDs are unvalidated strings; no official names/descriptions
3. **ASI IDs in Attack steps** — Need `asiIds` field like `owaspIds`

### Background

OWASP now maintains four Top 10 lists relevant to threat modeling:

| List | Year | Focus |
|------|------|-------|
| API Security Top 10 | 2023 | API vulnerabilities |
| LLM Applications Top 10 | 2025 | LLM-specific risks |
| Web Application Top 10 | 2021 | Traditional web vulnerabilities |
| **Agentic Applications Top 10 (ASI)** | 2026 | Autonomous AI agent risks |

The ASI list is particularly relevant for AI coding agents like OpenClaw, addressing risks like:

- ASI02: Tool Misuse & Exploitation
- ASI03: Agent Identity & Privilege Abuse
- ASI04: Agentic Supply Chain Compromise
- ASI05: Unexpected Code Execution
- ASI09: Human-Agent Trust Exploitation

// GetOWASPEntry returns the reference data for an OWASP ID, or nil if not found.
func GetOWASPEntry(id string) *OWASPEntry {
    if e, ok := OWASPAPITop10[id]; ok {
        return &e
    }
    if e, ok := OWASPLLMTop10[id]; ok {
        return &e
    }
    if e, ok := OWASPWebTop10[id]; ok {
        return &e
    }
    if e, ok := OWASPAgenticTop10[id]; ok {
        return &e
    }
    return nil
}

// ValidateOWASPID checks if an OWASP ID is recognized.
func ValidateOWASPID(id string) bool {
    return GetOWASPEntry(id) != nil
}

### 4. OWASP Validation

**Priority:** Medium

Add optional validation to check that OWASP IDs in mappings are recognized.

**Changes to `ir/validate.go`:**

```go
// ValidateOWASPMappings checks that OWASP IDs are recognized.
// Returns warnings (not errors) for unrecognized IDs.
func (d *DiagramIR) ValidateOWASPMappings() []string {
    var warnings []string

    // Check top-level OWASP mappings
    if d.Mappings != nil {
        for _, m := range d.Mappings.OWASP {
            if !ValidateOWASPID(m.ID) {
                warnings = append(warnings, fmt.Sprintf("unrecognized OWASP ID: %s", m.ID))
            }
        }
    }

    // Check attack step OWASP IDs
    for _, a := range d.Attacks {
        for _, id := range a.OWASPIds {
            if !ValidateOWASPID(id) {
                warnings = append(warnings, fmt.Sprintf("attack step %d: unrecognized OWASP ID: %s", a.Step, id))
            }
        }
        for _, id := range a.ASIIds {
            if !ValidateOWASPID(id) {
                warnings = append(warnings, fmt.Sprintf("attack step %d: unrecognized ASI ID: %s", a.Step, id))
            }
        }
    }

    return warnings
}
```
