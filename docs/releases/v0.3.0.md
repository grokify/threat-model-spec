# v0.3.0 Release Notes

**Release Date:** 2026-04-18

This release introduces agent specifications and a Claude Code plugin for AI-assisted threat model diagram creation, along with repository reorganization.

## Highlights

- Agent specifications and Claude Code plugin for threat model diagram creation
- Examples moved to separate repository for focused maintenance

## What's New

### Agent Specifications

New agent specifications for AI-assisted threat model visualization:

| Agent | Description |
|-------|-------------|
| `attack-flow-visualizer` | Creates D2 attack chain diagrams from threat model data |
| `diagram-quality-reviewer` | Reviews and improves diagram quality |
| `dfd-creator` | Creates data flow diagrams from threat models |

### Claude Code Plugin

A complete Claude Code plugin implementation including:

- **Agents** — Specialized agents for diagram creation and review
- **Commands** — `create-dfd` command for generating data flow diagrams
- **Skills** — Reusable skills for legend design, layout optimization, and numbered flow creation

Plugin location: `agents/plugins/claude/`

### Repository Reorganization

The examples directory has been removed. This allows the core library to focus on the specification and tooling.

## Installation

### Go Library

```bash
go get github.com/grokify/threat-model-spec@v0.3.0
```

### CLI Tool

```bash
go install github.com/grokify/threat-model-spec/cmd/tms@v0.3.0
```

## Requirements

- Go 1.24+
- [D2](https://d2lang.com) v0.6+ (for SVG rendering)
